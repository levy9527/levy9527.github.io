2018-10-29 总结

明天即将启航，再去北京

工作相关

回来除了休息，主要做以下事情

  1. 结合实践经验，出一份适合当前团队的前端笔试题，并建立题库
  2. 实践耗子哥的练级攻略
  3. 看完了Bob大叔《程序员职业素养》
  4. 跟实习生们碰了个面，并给他们发了培训资料

对于1，我升华的点在于：v神只是叫我出20道题，而我一是不想生搬硬套直接抄网上的，二是我觉得还有些看重的点只有20道并不能体现，于是我每一道都是结合实际，精心设计; 并出了不止20道题，留存起来，做为题库，以备不时之需。

另外，我还组织前端同学一起来做下题目，并加讨论。在这个过程中，我意识到，这是一个做练习的过程，也是一个教学相长的过程。根据我的知识储备，为了节约时间，也为了简洁，我的思考逻辑表示如下：

  1. 做习题 => 刻意练习，从新手到大师
  2. 学生时代结束后，做题就少了; 国外的书一大特点就是有习题 => 没有习题，就自己创造习题 => 就像AlphaGo自己跟自己对弈
  3. 教学相长 => 学习金字塔 => 教别人的学习效率最高
  4. 一年前的我不会这样思考，今天的我会这样思考 => 其中的秘诀是什么，这种提升是否可以复制 => 让其他人也觉醒起来

对于2，我已经意识到，每天必须抽出点时间用来学习，用于自我提升，所以必须每天坚持，就像2016年点亮oschina的贡献图一样。

对于3，集合了Bob大叔40余年的编程经验，告诉我们真正专业的程序员应当具备什么样的态度，原则，采取什么样的行动。刚好是百得利一期实战过后，带着疑问来看，醍醐灌顶。这就像做完魅族去看《人月神话》一样，豁然开朗。重新审视这次项目经历，当初思想觉悟不够，有许多地方做得不妥当，归根结底是不够专业。这本书我会梳理出思维导图，跟大伙儿分享。

对于4，我更愿意教内功心法，而不是招式。这点我跟耗子哥的想法是一致的：进入专业领域前，必须了解专业人员需要具备的素养; 素养不够，只能沦为码农。另外，我更愿意提供信息源，让他们自己去按图索骥，咀嚼消化，因为学习需要自我驱动，要学会自己“找食物”，而不是“等着喂”。

生活相关

另外，回来之后，我开始下厨了，这几天在广州的晚上，我都没下楼，都是自己煮的。

两个月前跟我讲这种事，我不会相信，毕业后我就没下过厨房; 但北京出差回来，我确实是这样改变了。

要问我原因，我也答不上来。我当然可以很理性地找出几点理由，但我觉得那都是经过加工的，不是真实原因。我更认为这是一种冥冥中的自我驱动。

以上帝视角来看，我，用着俊平的锅，胡栋的铲，给他们做早餐。这是一件很奇妙的事。

奇妙的点在于：

  1. 胡栋一个人住的时候是有做饭的，住一起后，我们幻想着胡栋做饭给我们吃，但从未变成现实。
  2. 俊平在大家的印象中也是很努力的，跟”懒”是挂不上钩的，但把锅放在厨房后，从来没洗过。
  3. 而我，一直都是出去吃，喜欢叫外卖的人，自己内心也从未觉得不下厨有什么不妥，因为我觉得柴米油盐锅碗瓢盆很麻烦，最终却成了下厨的先行者。

想不通原因就不想，确认这个事实，然后享受，并从中琢磨些东西，以下是我的脑回路：

  1. 与其寄期望于他人，不如自己行动起来，去带动别人
  2. 十二道锋味 => 做一个会下厨的男人
  3. 治大国如烹小鲜 => 老子一定是个厨艺高手 => 我更加认可老庄思想了
  4. 关注厨房智慧，这也是生活智慧的一部分

情绪波动

上面是自我提升的点，也是闪光点。

一直以来，我都是以正能量的形象出现，解决问题，答疑解惑，排忧解难，充满理性，偶尔浮夸一把，调戏一下小女生，很洒脱的样子，似乎没有让我烦心的事。

但其实，我也有弱点，我也有软肋，我也会困惑，也会紧张不安，也会焦虑不已，只不过一直都在忙着前进，大多时间睡一觉，就能把这些负面的东西压下去了。

但今天，让我很无力的是，我今天做不到风平浪静了。我想今天下午就订机票，也有一部分这个原因，想让自己忙起来，在路上，就能把别的事情抛在脑后。

但毕竟是明天的飞机，结果下午我坐立不安，根本无心编码。心不在焉把出差报销单填完，再帮实习生出个培训资料，已经是极限了。

终究还是有人能够轻易左右我的情绪，让我起波澜。

我感到乏力。

情绪是不理性的，没办法用思考与逻辑去征服。

以前在云徙的时候就表现过不理智的一面，就像个孩子一样。今天的我虽然表面看来寻常无事，但实际根本不在状态。虽然有进步，但还不够。这不是我想到的样子。

所以下午早早回去。觉得提不起劲，很不在状态，而这也是晚上看《模仿游戏》落泪的一个原因。要是正常状态下，我会钦佩图灵的行为，为之感动，却也不至于因为同情，看他流泪而流泪。 其实是情绪共振，因而有相同的行为罢了。

有个词叫不吐不快，以上心理活动，我闷在心理会发酵，会一直难受，这驱使我很真实地写出来。

写到这里，心里舒畅多了。想起冷爱的经历，心里难受，就会写出来，他说，“这可能是具备作家潜力的一个特质”。哈哈，我初中的时候也是想当作家的，看来还有那么些道理。

总而言之，心态要调整好，不然在公司却无心工作，那就很不专业了。
